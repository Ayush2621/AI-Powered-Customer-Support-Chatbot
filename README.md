# AI-Powered Customer Support Chatbot

An intelligent, contextual FAQ and customer support chatbot backend built using Python, FastAPI, Hugging Face Transformers, and SQLite.

## 🚀 Features
- **Contextual AI Responses:** Powered by the open-source `TinyLlama-1.1B` chat model.
- **Robust API Layer:** Built using FastAPI with auto-generated documentation.
- **Persistent Data Logging:** Tracks all user interactions and bot responses in an SQLite database.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **NLP Engine:** Hugging Face Transformers & PyTorch
- **Backend Framework:** FastAPI & Uvicorn
- **Database:** SQLite3
